# Atlas Agents

- Manager Agent: coordinates tasks.
- Planner Agent: decomposes user goals.
- Browser Agent: controls browser workflows.
- Vision Agent: analyzes screenshots and images.
- Writer Agent: drafts user-facing documents.
- Reviewer Agent: checks quality and risk.
- Memory Agent: stores reusable project context.
- Marketing Agent: creates listing and campaign copy.
- Resume Agent: tailors existing resume content without inventing experience.
- Cover Letter Agent: drafts role-specific letters.
- Job Analyzer Agent: scores job descriptions.
- Visa Intelligence Agent: detects sponsorship risk.
- Video Planner Agent: creates property/storyboard plans.
- Property Analyzer Agent: evaluates property listings.
